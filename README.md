# Customer-Sentiment-Classification-at-Disneyland
This project is done by using Google Colaboratory and tools in NLTK libraries (Natural Language Toolkit): VADER (Valence Aware Dictionary and Sentiment Reasoner) and textBlob.

Download the dataset here: https://www.kaggle.com/datasets/arushchillar/disneyland-reviews

# Summarize
<p align="justify">“Customer Sentiment Classification at Disneyland Using Machine Learning Models” was conducted with the aim of developing an automated system capable of analyzing and classifying customer sentiments based on reviews and feedback collected from online platforms. Identifying customer sentiments helps organizations better understand customer satisfaction levels, thereby supporting service improvement and informed decision-making.</p>

# About this project
<p align="justify">In this project, customer reviews from Disneyland were collected and preprocessed using Natural Language Processing (NLP) techniques. The textual data were then transformed into numerical features for training machine learning models to classify sentiments into categories such as positive, negative, and neutral. The experimental process included data preprocessing, feature extraction, model training, and performance evaluation using metrics such as Accuracy, Precision, Recall, and F1-Score.</p>

<p align="justify">The experimental results demonstrate that machine learning models can effectively identify and classify customer sentiments with high accuracy. These findings highlight the potential of machine learning in customer sentiment analysis and its usefulness for evaluating service quality at Disneyland. Furthermore, the proposed approach can be extended to various domains, including tourism, e-commerce, and customer service applications.</p>

<p align="justify">The prediction on unseen data was performed to evaluate the model's generalization capability. By testing the model on reviews that were not included in the training process, we can assess its ability to classify customer sentiments accurately in real-world scenarios. This step ensures that the model learns meaningful patterns from the data rather than merely memorizing the training samples.</p>

# Machine Learning algorithms used in this project: 
Bernoulli Naive Bayes, Gaussian Naive Bayes, DecisionTreeClassifier, RandomForestClassifier, LogisticRegression
